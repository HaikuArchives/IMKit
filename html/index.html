Purpose

The purpose of the Instant Messaging Kit (IM kit) is to provide a uniform way to access the various IM networks that exist, and to provide a connection between a People file and the IM accounts that person has.

The IM Kit is a modular framework developed to make it easy to access various IM networks (ICQ, AIM, etc) and is designed in a way it is fully integrated with BeOS. It makes use of BeOS attributes, indexes and live queries to make it as flexible as possible. With the IM Kit you can, for instance, search for all members of your family that are online and that by using the standard Find interface. You can also manage all the contacts using Tracker.

What's in it right now

The im_server, a protocol add-on, and a sample client.

Current functionality includes:

  • Connection to ICQ
  • Connection to AIM
  • People file <> IM connection
  • IM-specific People file handing add-ons for the Tracker
  • Simple chat client
  • Simple logger
  • Probably lots of bugs and strange behaviour
